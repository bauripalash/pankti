# সাধারণ লাইব্রেরি - ওএস

পঙক্তির সাধারণ লাইব্রেরির মধ্যে বর্তমান **ওএস** লাইব্রেরিতে বেশ কয়েকটি ফাংশন আছে যাদের দ্বারা বর্তমান অপারেটিং সিস্টেম সম্বন্ধে নানারকম তথ্য পাওয়া যায়।

```pankti
আনয়ন ওএস "ওএস"
```
