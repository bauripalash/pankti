# সাধারণ লাইব্রেরি - পরিবেশ

সিস্টেমের বিভিন্ন তথ্য যেমন অপারেটিং সিস্টেমের নাম, আর্কিটেকচার, ব্যবহারকারী ইত্যাদি জানার কাজ এই উৎসের মধ্যে রয়েছে। এই উৎসটি কোনো স্ক্রিপ্টে এইভাবে আনতে হয় - 

```pankti
আনয়ন পরিবেশ "পরিবেশ"
```

# সাধারণ কাজসমূহ
## পরিবেশ.**নাম()**
এই কাজটি বর্তমান অপারেটিং সিস্টেমের নাম কথারাশি হিসাবে ফেরত দেয়। সম্ভাব্য ফেরত মানগুলি হলঃ `লিনাক্স`, `উইন্ডোজ`, `ম্যাকওএস`, `ওয়েব`। তবে এইগুলির বাইরে কোনো অপারেটিং সিস্টেম হলে `অজানা` ফেরত দেবে।
###### **উদাহরণ** {.dh3}
```pankti
আনয়ন পরিবেশ "পরিবেশ"
?পরিবেশ.নাম()
```
## পরিবেশ.**আর্চ()**
এই কাজটি বর্তমান অপারেটিং সিস্টেমের আর্কিটেকচার কথারাশি হিসাবে ফেরত দেয়। সম্ভাব্য ফেরত মানগুলি হলঃ `আর্ম`, `এক্স৮৬`, `এক্স৮৬_৬৪`। তবে এর বাইরে অন্য কোন আর্কিটেকচার হলে `অজানা` ফেরত দেবে। 
###### **উদাহরণ** {.dh3}
```pankti
আনয়ন পরিবেশ "পরিবেশ"
?পরিবেশ.আর্চ()
```
## পরিবেশ.**ব্যবহারকারী()**
এই কাজটি বর্তমান ব্যবহারকারীর *Username* কথারাশি হিসাবে ফেরত দেয়। অনলাইন সংস্করণে এটি `ওয়েব` ফেরত দেবে। এছাড়া কোনো কারণে *Username* জানা না গেলে `অজানা` ফেরত দেবে।
###### **উদাহরণ** {.dh3}
```pankti
আনয়ন পরিবেশ "পরিবেশ"
?পরিবেশ.ব্যবহারকারী()
```
## পরিবেশ.**ঘর()**
এই কাজটি বর্তমান ব্যবহারকারীর *Home Directory* কথারাশি হিসাবে ফেরত দেয়। অনলাইন সংস্করণে এটি `ওয়েব` ফেরত দেবে। এছাড়া কোনো কারণে *Home Directory* জানা না গেলে `অজানা` ফেরত দেবে।
###### **উদাহরণ** {.dh3}
```pankti
আনয়ন পরিবেশ "পরিবেশ"
?পরিবেশ.ঘর()
```

## পরিবেশ.**অবস্থান()**
এই কাজটি বর্তমান ব্যবহারকারীর *Current Directory* অর্থাৎ সে ফোল্ডার/ডাইরেক্টরি থেকে প্রোগ্রামটি চালানো হচ্ছে সেটি কথারাশি হিসাবে ফেরত দেয়। অনলাইন সংস্করণে এটি `ওয়েব` ফেরত দেবে। এছাড়া কোনো কারণে *Current Directory* জানা না গেলে `অজানা` ফেরত দেবে।
###### **উদাহরণ** {.dh3}
```pankti
আনয়ন পরিবেশ "পরিবেশ"
```
