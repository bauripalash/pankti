# সাধারণ লাইব্রেরি - পট
গ্রাফিক্স সম্পর্কিত কাজের জন্য এই উৎসে অনেকগুলি কাজ বর্তমান। এই গ্রাফিক্স উৎস দিয়ে গেম তৈরি, দৃশ্যমান প্রোগ্রাম তৈরি ইত্যাদি কাজ সহজ করে তোলা হয়েছে। এই উৎসটি কোনো স্ক্রিপ্টে এইভাবে আনতে হয় - 

```pankti
আনয়ন পট "পট"
```

## পট.**নতুন(*পর্দার\_দৈর্ঘ্য*, *পর্দার\_প্রস্থ*, *পর্দার\_শিরোনাম*)** {.stdfunc}
পটচিত্রের মূল পর্দা/Window তৈরি করার জন্য এই কাজটি প্রয়োজনীয় প্রস্তুতি সম্পূর্ণ করে। পর্দার দৈর্ঘ্য, পর্দার প্রস্থ প্রেরণমানগুলি সংখ্যা হবে এবং পর্দার শিরোনাম একটি কথারাশি হবে। 


###### **উদাহরণ** {.dh3}
```pankti
পট.নতুন(800, 600, "পট")
```

## পট.**বন্ধ()** {.stdfunc}
পটচিত্রের মূল পর্দা/Window বন্ধ করার জন্য এই কাজটি ব্যবহার করা হয়।

###### **উদাহরণ** {.dh3}
```pankti
পট.বন্ধ()
```

## পট.**চলমান()** {.stdfunc}
পটের মূল পর্দা/Window ব্যবহারকারী যতক্ষণ না বন্ধ করে ততক্ষণ এই কাজটি `সত্যি` ফেরত দেবে। 

সাধারণত গেমের প্রধান লুপের শর্ত হিসাবে ব্যবহার করা হয়। 

###### **উদাহরণ** {.dh3}
```pankti
যতক্ষণ পট.চলমান() করো
    // এখানে মূল অঙ্কন এবং কিবোর্ড ইত্যাদি নিয়ে কাজ করা হবে
শেষ
```

## পট.**আঁকা\_শুরু()** {.stdfunc}
প্রতিটি ফ্রেমের শুরুতে এই কাজটি চালাতে হবে যাতে পর্দার মধ্যে এর পরবর্তী অঙ্কনের কাজগুলি সঠিকভাবে কাজ করতে পারে। ডেল্টা সময় এই কাজের মধ্যেই অভ্যন্তরীণ ভাবে গণনা করা হয়। `পট.আঁকা_শুরু()` এবং `পট.আঁকা_শেষ()` এই দুই কাজের মধ্যে অঙ্কনের কাজগুলি লিখলেই তবে সেগুলি প্রতি ফ্রেমে দৃশ্যমান হবে।

###### **উদাহরণ** {.dh3}
```pankti
পট.আঁকা_শুরু()
// এই দুই কাজের মধ্যে সমস্ত অঙ্কনের কাজগুলি থাকবে
পট.আঁকা_শেষ()
```

## পট.**আঁকা\_শেষ()** {.stdfunc}
প্রতিটি ফ্রেমের শেষ এই কাজটি চালাতে হবে যাতে পূর্ববর্তী অঙ্কনের কাজগুলি পর্দাতে দেখানো হয়।

###### **উদাহরণ** {.dh3}
```pankti
পট.আঁকা_শুরু()
// এই দুই কাজের মধ্যে সমস্ত অঙ্কনের কাজগুলি থাকবে
পট.আঁকা_শেষ()
```
## পট.**পরিষ্কার()** {.stdfunc}
এই কাজ পর্দাটিতে এই কাজের ডাকের আগে পর্যন্ত যত কিছু আঁকা হয়েছে তা সব মুছে ফেলে। সাধারণত প্রতিটি ফ্রেমের শুরুতে এই কাজটি ব্যবহার করে পর্দাটি পরিষ্কার করে নেওয়া হয়। 

###### **উদাহরণ** {.dh3}
এই নির্দেশলিপিটি চালালে একটি ফাঁকা সাদা পর্দা দেখাবে
```pankti
আনয়ন পট "পট"
পট.নতুন(৫০০, ৫০০, "পঙক্তি পর্দা")
যতক্ষণ পট.চলমান() করো
    পট.পরিষ্কার()
    পট.আঁকা_শুরু()
    পট.আঁকা_শেষ()
শেষ
পট.বন্ধ()
```
