# সাধারণ উৎস - **তালিকা**

তালিকা সম্পর্কিত বেশ কিছু সাধারণ কাজ এই উৎসের মধ্যে বর্তমান। এই উৎসটি কোনো স্ক্রিপ্টে এইভাবে আনতে হয় - 
```pankti
আনয়ন তালিকা "তালিকা"
```

# সাধারণ কাজসমূহ

## তালিকা.**বর্তমান(*তালিকার_নাম*, *উপাদান*)** {.stdfunc}

যদি উপাদানটি তালিকার মধ্যে বর্তমান থাকে, তাহলে এই কাজটি **সত্যি** ফেরত দেয় নাহলে **মিথ্যা** ফেরত দেয়। 

> [!WARNING]
> * তালিকার_নাম একটি তালিকা হবে 
> * সূচক যে কোনো রাশি হবে

###### **উদাহরণ** {.dh3}
```pankti
আনয়ন তালিকা "তালিকা"
ধরি ফলের_তালিকা = ["আম", "জাম", "আপেল", "আনারস"]
তালিকা.বর্তমান(ফলের_তালিকা, "বেগুন") // মিথ্যা
তালিকা.বর্তমান(ফলের_তালিকা, "আপেল") // সত্যি
```
## তালিকা.**সংযোগ(*তালিকার_নাম*, *সূচক*, *উপাদান*)** {.stdfunc}

তালিকাতে সূচক স্থানে উপাদানকে যোগ করে এবং তালিকার নতুন আয়তন ফেরত দেয়। 

> [!WARNING]
> * তালিকার_নাম একটি তালিকা হবে
> * সূচক একটি ধনাত্মক পূর্ণসংখ্যা হবে
> * উপাদান যে কোনো রাশি হবে

###### **উদাহরণ** {.dh3}
যদি একটি তালিকা থাকে
```pankti
ধরি ফলের_তালিকা = ["আম", "জাম", "আনারস"]
```
এখানে `আম`-এর সূচক হচ্ছে ০, `জাম`-এর সূচক ১ এবং `আনারস`-এর সূচক হচ্ছে ২। এবার যদি আমরা ৩ নং অবস্থান অর্থাৎ ২ নং সুচকে `আপেল` কে ঢোকাতে চাই তাহলে আমরা এই `সংযোগ(...)` কাজটি ব্যবহার করবো। 
```pankti
তালিকা.সংযোগ(ফলের_তালিকা, ২, "আপেল")
```
এটি করলে `ফলের_তালিকা` কিছুটা এইরকম দেখতে হবে
```pankti
[আম, জাম, আপেল, আনারস]
```
## তালিকা.**সূচক(*তালিকার_নাম*, *উপাদান*)** {.stdfunc}

> [!WARNING]
> * তালিকার_নাম একটি তালিকা হবে
> * উপাদান যে কোনো রাশি হবে

তালিকার মধ্যে যদি উপাদান বর্তমান থাকে তাহলে তার সূচক ফেরত দেয় আর নাহলে -১ ফেরত দেয়। 

###### **উদাহরণ** {.dh3}
আমাদের ফলের তালিকায় যদি আমরা `আনারস` -এর সূচক জানতে চাই, তাহলে এই কাজটি ব্যবহার করবো। 
```pankti
ধরি ফলের_তালিকা = ["আম", "জাম", "আপেল", "আনারস"]
?তালিকা.সূচক(ফলের_তালিকা, "আনারস")
```
এটি চালালে উত্তর দেখাবে ৩ কারণ, আনারস ৪নং স্থানে অর্থাৎ ৩নং সুচকে অবস্থিত।

## তালিকা.**বিয়োগ(*তালিকার_নাম*, *সূচক*)** {.stdfunc}

> [!WARNING]
> * তালিকার_নাম একটি তালিকা হবে
> * সূচক একটি ধনাত্মক পূর্ণসংখ্যা হবে
তালিকার মধ্যে থেকে প্রদত্ত সুচকে থাকা উপাদানটি তালিকা থেকে বাদ দিয়ে সেটি ফেরত দেয়।

###### **উদাহরণ** {.dh3}
আমাদের ফলের তালিকায় আমরা ভুল করে ফলের নামের সাথে একটি ফুলের নাম লিখে ফেলেছি। 
```pankti
ধরি ফলের_তালিকা = ["আম", "গোলাপ", "জাম", "আপেল", "আনারস"]
```
আমাদের এই তালিকা থেকে "গোলাপ" উপাদনটি তালিকা থেকে মুছে ফেলার জন্য আমার এই `বিয়োগ(...)` কাজটি ব্যবহার করবো। এই তালিকায় `গোলাপ` আছে দ্বিতীয় স্থানে অর্থাৎ ১ নং সূচকে। তাই আমরা এইভাবে কাজটি ব্যবহার করবোঃ

```pankti
তালিকা.বিয়োগ(ফলের_তালিকা, ১)
```
এবার যদি আমরা তালিকা দেখি তাহলে দেখবো তার বর্তমান অবস্থা কিছুটা এইরকম

```pankti
[আম, জাম, আপেল, আনারস]
```

এই কাজটি বাদ হওয়া উপাদানটি ফেরত দেয়, তাই যদি আমরা এইভাবে লিখতাম, তাহলে `বাদ` চলরাশির মান হতো `গোলাপ`
```pankti
ধরি বাদ = তালিকা.বিয়োগ(ফলের_তালিকা, ১)
```
## তালিকা.**শেষবাদ(*তালিকার_নাম*)** {.stdfunc}

> [!WARNING]
> * তালিকার_নাম একটি তালিকা হবে

এই কাজটি তালিকার একেবারে শেষের উপাদানটি মুছে দিয়ে সেই মুছে যাওয়া উপাদানটি ফেরত দেয়।

###### **উদাহরণ** {.dh3}
আমাদের ফলের তালিকার শেষ মানটি বাদ দেওয়ার জন্য এই কাজটি ব্যবহার করবোঃ 

```pankti
ধরি ফলের_তালিকা = ["আম", "জাম", "আপেল", "আনারস"]
তালিকা.শেষবাদ(ফলের_তালিকা)
```

এরপর ফলের_তালিকার বর্তমান অবস্থা হবে এই রকমঃ
```
[আম, জাম, আপেল]
```
বিয়োগের মতো এই কাজটি বাদ হয়ে হওয়া উপাদানটি ফেরত দেয় অর্থাৎ, আমরা যদি এইভাবে লিখতাম তাহলে `বাদ` চলরাশির মান হতো `আনারস
```pankti
ধরি বাদ = তালিকা.শেষবাদ(ফলের_তালিকা)
```
