# সাধারণ লাইব্রেরি - ছক

ছক সম্পর্কিত বেশ কিছু সাধারণ কাজ এই উৎসের মধ্যে বর্তমান। এই উৎসটি কোনো স্ক্রিপ্টে এইভাবে আনতে হয় -

```pankti
আনয়ন ছক "ছক"
```

# সাধারণ কাজসমূহ
## ছক.**বর্তমান(*ছকের_নাম*, *সূচক*)** {.stdfunc}
এই কাজটি যাচাই করে `সূচক` কি `ছকের_নাম` -এর মধ্যে বর্তমান। যদি বর্তমান থাকে তাহলে সত্যি ফেরত দেয় নাহলে মিথ্যা। 

|প্রেরণমান|ধরন|বিবরণ|
|------|--|----|
|ছকের_নাম|ছক|যে ছকে সূচকটি খোঁজা হবে|
|সূচক|যেকোনো রাশি|যে সূচকটি খোঁজা হবে|
{.args-table}

|ফেরতমানের ধরন|বিবরণ|
|-----------|----|
|সত্যমান|-|
{.return-table}

###### **উদাহরণ** {.dh3}
ধরি একটি ছক আছেঃ 
```pankti
ধরি তথ্য = { "দিন" : "রবিবার", "মাস" : "বৈশাখ", "খ্রিস্টাব্দ" : ২০২৬ } 
```
এই উপাদানের মধ্যে তিনটি উপাদান আছে, এবার যদি আমরা জানতে চাই এই ছকের মধ্যে কি "দিন" নামের কোনো উপাদান আছে তখন এই কাজটি ব্যবহার করবোঃ
```pankti
ছক.বর্তমান(তথ্য, "দিন")
```
এই রাশিমালার মান হবে সত্যি কারণ `দিন` নামের একটি সূচক এর মধ্যে আছে। আবার যদি আমরা জানার চেষ্টা করি `সময়` নামের কোনো সূচক আছে কি না, তখন এইভাবে লিখব
```pankti
ছক.বর্তমান(তথ্য, "সময়")
```
এইবার কিন্তু এই রাশিমালার মান হবে `মিথ্যা`। কারণ উপরোক্ত `তথ্য` নামের ছকে সময় বলে কোনো সূচক নেই।

## ছক.**বিয়োগ(*ছকের_নাম*, *সূচক*)** {.stdfunc}

|প্রেরণমান|ধরন|বিবরণ|
|------|--|----|
|ছকের_নাম|ছক|যে ছক থেকে সূচকটি মুছে ফেলা হবে|
|সূচক|যেকোনো রাশি|যে সূচকটির সূচক-মান যুগল মুছে ফেলা হবে|
{.args-table}
|ফেরতমানের ধরন|বিবরণ|
|-----------|----|
|যেকোনো রাশি|যে সূচকটি মুছে ফেলা হল তার মান|
{.return-table}


এই কাজটি `ছকের_নাম` থেকে সূচক -এর `সূচক : মান` যুগলটি ছক থেকে মুছে ফেলে, এবং মুছে যাওয়া সূচকের মানটি ফেরত দেয়। যদি ছকের মধ্যে প্রদত্ত সূচক বর্তমান না থাকে তাহলে গোলযোগ দেখাবে। 
###### **উদাহরণ** {.dh3}
ধরি একটি ছক আছেঃ 
```pankti
ধরি তথ্য = { "দিন" : "রবিবার", "মাস" : "বৈশাখ", "খ্রিস্টাব্দ" : ২০২৬ } 
```
এবার যদি এই ছক থেকে খ্রিস্টাব্দের সূচক:মান যুগলটি বাদ দিতে চাই তাহলে এইভাবে কাজটি ব্যবহার করবোঃ

```pankti
ছক.বিয়োগ(তথ্য, "খ্রিস্টাব্দ")
```

এরপর যদি এই `তথ্য` ছকটি দেখার চেষ্টা করি তাহলে এইরকম দেখতে পাবো
```pankti
?তথ্য
```
এটা চালালে দেখাবেঃ
```pankti
{দিন : রবিবার, মাস : বৈশাখ}
```
