# সাধারণ লাইব্রেরি - ছক

পঙক্তির সাধারণ লাইব্রেরির মধ্যে বর্তমান **ছক** লাইব্রেরিতে বেশ কয়েকটি ফাংশন আছে যাদের দ্বারা পঙক্তির ছক প্রকৃতির রাশি নিয়ে নিয়ে বিভিন্ন কাজ করা যায়।

```pankti
আনয়ন ছক "ছক"
```
