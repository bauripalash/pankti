# সাধারণ লাইব্রেরি - গণিত

পঙক্তির সাধারণ উৎস **গণিত** -এর মধ্যে কিছু অতিপরিচিত গণিত সংক্রান্ত কাজ এবং ধ্রুবক আছে। এই উৎসটি ব্যবহার করার জন্য প্রথমে এটিকে স্ক্রিপ্টের মধ্যে আনতে হবে।

```pankti
আনয়ন গণিত "গণিত"
```

এই কাজটি `ক` এর বর্গমূল ফেরত দেয়।

> [!INFO]
> গাণিতিক সমীকরণঃ 
> **উত্তর** = \\( \sqrt{ক} \\)

###### **উদাহরণ** {.dh3}
```pankti
আনয়ন গণিত "গণিত"
?গণিত.বর্গমূল(২৫) // 5
?গণিত.বর্গমূল(১০) // 3.162278
?গণিত.বর্গমূল(৯৮০১) // 99
```

###### **উদাহরণ** {.dh3}
```pankti

?গণিত.সংখ্যা("-১০০.৫")
?গণিত.সংখ্যা("১০০১.১")
```


## গণিত.**পরম(*ক*)** {.stdfunc}
এই কাজটি ক -এর পরম মান ফেরত দেয়। অর্থাৎ ক এর মান ঋণাত্মক হলেও ধনাত্মক মান ফেরত দেবে। 
###### **উদাহরণ** {.dh3}
```pankti
আনয়ন গণিত "গণিত"
?গণিত.পরম(১০০)      // ফলাফলঃ ১০০
?গণিত.পরম(-১০০)     // ফলাফলঃ ১০০
?গণিত.পরম(০.৫)      // ফলাফলঃ ০.৫
?গণিত.পরম(-০.৫)     // ফলাফলঃ ০.৫
``` 

## গণিত.**আসন্ন(*ক*)** {.stdfunc}
এটি ক এর আসন্ন মান ফেরত দেয়। ক এর দশমিক স্থানীয় অংশ যদি `.৫` -এর বেশি হয় তাহলে `ক + ১` অর্থাৎ পরবর্তী বড় পূর্ণসংখ্যা ফেরত দেয় নাহলে শুধু ক ফেরত দেয়। 

>[!TIP]
> **বিঃদ্রঃ** এই কাজটি অন্যান্য প্রোগ্রামিং ভাষাতে থাকা round(ক) কাজের সমতুল্য। 
>

###### **উদাহরণ** {.dh3}
যদি ক এর মান হয় ১.৫ তাহলে এই কাজ উত্তর দেবে ২। আবার যদি ক এর মান ১.৪ হয় তাহলে এই কাজ উত্তর দেবে ১। কয়েকটি উদাহরণ হলঃ 
```pankti
আনয়ন গণিত "গণিত"
?গণিত.আসন্ন(১)      // ফলাফলঃ ১
?গণিত.আসন্ন(১.৪)    // ফলাফলঃ ১
?গণিত.আসন্ন(১.৫)    // ফলাফলঃ ২
?গণিত.আসন্ন(১.৬)    // ফলাফলঃ ২
?গণিত.আসন্ন(১.৪৫)   // ফলাফলঃ ১
?গণিত.আসন্ন(১.৫৬)   // ফলাফলঃ ২
?গণিত.আসন্ন(১.৯৯)   // ফলাফলঃ ২
```

## গণিত.**আসন্ন_ছোট(*ক*)** {.stdfunc}
এই কাজটি ক এর দশমিক স্থানীয় সংখ্যাগুলি বাদ দিয়ে পূর্ণসংখ্যাটি ফেরত দেয়। ক এর দশমিক স্থানীয় মান .৫ বা .৪ এমনকি .৯৯৯৯ হলেও এই কাজটি ক-এর মুল পুর্নসংখ্যাটি ফেরত ফেরত। 

>[!TIP]
> **বিঃদ্রঃ** এই কাজটি অন্যান্য প্রোগ্রামিং ভাষাতে থাকা floor(ক) কাজের সমতুল্য। 
>

###### **উদাহরণ** {.dh3}
`আসন্ন(...)` কাজের উদাহরণের দেওয়া সংখ্যাগুলি যদি এই কাজে ব্যবহার করি তাহলে ফলাফল কিছুটা এইরকম হবেঃ

```pankti
আনয়ন গণিত "গণিত"
?গণিত.আসন্ন_ছোট(১)      // ফলাফলঃ ১
?গণিত.আসন্ন_ছোট(১.৪)    // ফলাফলঃ ১
?গণিত.আসন্ন_ছোট(১.৫)    // ফলাফলঃ ১
?গণিত.আসন্ন_ছোট(১.৬)    // ফলাফলঃ ১
?গণিত.আসন্ন_ছোট(১.৪৫)   // ফলাফলঃ ১
?গণিত.আসন্ন_ছোট(১.৫৬)   // ফলাফলঃ ১
?গণিত.আসন্ন_ছোট(১.৯৯)   // ফলাফলঃ ১
```

## গণিত.**আসন্ন_বড়(*ক*)** {.stdfunc}
এই কাজটিতে দেওয়া সংখ্যা `ক` এর মধ্যে যদি যেকোনো দশমিক অংশ বর্তমান থাকে তাহলে পরবর্তী পূর্ণসংখ্যা অর্থাৎ `ক + ১` -এর মান ফেরত দেবে। যদি ক পুর্নসংখ্যা হয় তাহলে `ক` -এর মানই ফেরত দেবে। 

>[!TIP]
> **বিঃদ্রঃ** এই কাজটি অন্যান্য প্রোগ্রামিং ভাষাতে থাকা ceil(ক) কাজের সমতুল্য। 
>

###### **উদাহরণ** {.dh3}
`আসন্ন(...)` কাজের উদাহরণের দেওয়া সংখ্যাগুলি যদি এই কাজে ব্যবহার করি তাহলে ফলাফল কিছুটা এইরকম হবেঃ
```pankti
আনয়ন গণিত "গণিত"
?গণিত.আসন্ন_বড়(১)      // ফলাফলঃ ১
?গণিত.আসন্ন_বড়(১.৪)    // ফলাফলঃ ১
?গণিত.আসন্ন_বড়(১.৫)    // ফলাফলঃ ১
?গণিত.আসন্ন_বড়(১.৬)    // ফলাফলঃ ১
?গণিত.আসন্ন_বড়(১.৪৫)   // ফলাফলঃ ১
?গণিত.আসন্ন_বড়(১.৫৬)   // ফলাফলঃ ১
?গণিত.আসন্ন_বড়(১.৯৯)   // ফলাফলঃ ১

// অন্য উদাহরণ
?গণিত.আসন্ন_বড়(৫)                   // ফলাফলঃ ৫
?গণিত.আসন্ন_বড়(৫.০০০০০৫)    // ফলাফলঃ ৬
?গণিত.আসন্ন_বড়(৫.৯৯৯৯৯৯)    // ফলাফলঃ ৬
```

## গণিত.**এলোমেলো(*ক*, *খ*)** {.stdfunc}
এই কাজটি ক থেকে খ এর মধ্যে যেকোনো একটি এলোমেলো দশমিক সংখ্যা ফেরত দেয়। যেহেতু এটি ভিন্ন ভিন্ন এলোমেলো সংখ্যা ফেরত দেয় তাই এমন হতেই পারে, একই স্ক্রিপ্ট চালানোর ফলে একই সংখ্যা বারবার ফেরত দেয়। 

>[!TIP]
> **বিঃদ্রঃ** এই কাজের দেওয়া মান দশমিক সংখ্যায় হবে, পূর্ণসংখ্যায় মান পাওয়ার জন্য `গণিত.আসন্ন(ক)` বা `গণিত.আসন্ন_ছোট(ক)` কিংবা `গণিত.আসন্ন_বড়(ক)` ব্যবহার করা যেতে পারে। 

###### **উদাহরণ** {.dh3}
এই স্ক্রিপ্টটি চালালে প্রতিবার ভিন্ন ভিন্ন মান দেখাবে। 
```pankti
আনয়ন গণিত "গণিত"
?গণিত.এলোমেলো(১, ১০)  // ১ থেকে ১০ এর মধ্যে (প্রতিবারই দশমিক সংখ্যা হবে)
?গণিত.এলোমেলো(০, ১)   // ০ থেকে ১ এর মধ্যে (প্রতিবারই দশমিক সংখ্যা হবে)
```
পূর্ণসংখ্যায় উত্তর পাওয়ার জন্য `গণিত.আসন্ন(ক)` কাজটি ব্যবহার করা যেতে পারে
```pankti
আনয়ন গণিত "গণিত"
?গণিত.আসন্ন(গণিত.এলোমেলো(১, ১০))
?গণিত.আসন্ন(গণিত.এলোমেলো(৫, ৫৫))
```

###### **উদাহরণ** {.dh3}
