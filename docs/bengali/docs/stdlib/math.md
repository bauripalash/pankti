# সাধারণ লাইব্রেরি - গণিত

পঙক্তির সাধারণ লাইব্রেরির মধ্যে বর্তমান **গণিত** লাইব্রেরিতে কিছু অতিপরিচিত গণিত সংক্রান্ত ফাংশন এবং ধ্রুবক আছে। এই মডিউলটি ব্যবহার করার জন্য প্রথমে এটিকে স্ক্রিপ্টের মধ্যে আনতে হবে।

```pankti
আনয়ন গণিত "গণিত"
```
