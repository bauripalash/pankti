# সাধারণ লাইব্রেরি - গণিত

পঙক্তির সাধারণ উৎস **গণিত** -এর মধ্যে কিছু অতিপরিচিত গণিত সংক্রান্ত কাজ এবং ধ্রুবক আছে। এই উৎসটি ব্যবহার করার জন্য প্রথমে এটিকে স্ক্রিপ্টের মধ্যে আনতে হবে।

```pankti
আনয়ন গণিত "গণিত"
```

# ধ্রুবকসমূহ

## গণিত.**পাই** {.stdfunc}
এই ধ্রুবকটি **পাই** (\\( \pi \\)) -এর নির্দিষ্ট মান ফেরত দেয় যা হল `৩.১৪১৫৯২৬৫৩৫৮৯৭৯৩`। 

###### **উদাহরণ** {.dh3}
```pankti
আনয়ন গণিত "গণিত"
?গণিত.পাই   // ফলাফলঃ ৩.১৪১৫৯২৬৫৩৫৮৯৭৯৩
```
## গণিত.**ই** {.stdfunc}
এই ধ্রুবকটি অয়লারের সংখ্যা (\\( e \\)) -এর একটি নির্দিষ্ট মান ফেরত দেয় যা হল `২.৭১৮২৮১৮২৮৪৫৯০৪৫`।
###### **উদাহরণ** {.dh3}

## গণিত.**এলোমেলো(*ক*, *খ*)** {.stdfunc}
এই কাজটি `ক` থেকে `খ` এর মধ্যে যেকোনো একটি এলোমেলো দশমিক সংখ্যা ফেরত দেয়। 

যেহেতু এটি ভিন্ন ভিন্ন এলোমেলো সংখ্যা ফেরত দেয় তাই একই স্ক্রিপ্ট একাধিকবার চালালে ভিন্ন ভিন্ন মান ফেরত দিতে পারে। তবে এমন হতেই পারে কাকতালীয়ভাবে একই সংখ্যা বারবার ফেরত আসতে পারে। 

|প্রেরণমান|ধরন|বিবরণ|
|------|--|----|
|ক|সংখ্যা|-|
|খ|সংখ্যা|-|
{.args-table}
|ফেরতমানের ধরন|বিবরণ|
|-----------|----|
|সংখ্যা|-|
{.return-table}

>[!TIP]
> **বিঃদ্রঃ** এই কাজের দেওয়া মান দশমিক সংখ্যায় হবে, পূর্ণসংখ্যায় মান পাওয়ার জন্য `গণিত.নিকটতম(ক)` বা `গণিত.নিকট_ছোট(ক)` কিংবা `গণিত.নিকট_বড়(ক)` ব্যবহার করা যেতে পারে। 

###### **উদাহরণ** {.dh3}
এই স্ক্রিপ্টটি চালালে প্রতিবার ভিন্ন ভিন্ন মান দেখাবে। 
```pankti
আনয়ন গণিত "গণিত"
?গণিত.এলোমেলো(১, ১০)  // ১ থেকে ১০ এর মধ্যে (প্রতিবারই দশমিক সংখ্যা হবে)
?গণিত.এলোমেলো(০, ১)   // ০ থেকে ১ এর মধ্যে (প্রতিবারই দশমিক সংখ্যা হবে)
```
পূর্ণসংখ্যায় উত্তর পাওয়ার জন্য `গণিত.নিকটতম(ক)` কাজটি ব্যবহার করা যেতে পারে
```pankti

?গণিত.নিকটতম(গণিত.এলোমেলো(১, ১০))
?গণিত.নিকটতম(গণিত.এলোমেলো(৫, ৫৫))
```
