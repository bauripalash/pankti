# সাধারণ লাইব্রেরি - কথা

পঙক্তির সাধারণ লাইব্রেরির মধ্যে বর্তমান **কথা** লাইব্রেরিতে বেশ কয়েকটি কাজ আছে যাদের দ্বারা কথারাশি নিয়ে বিভিন্ন কাজ করা যায়।

```pankti
আনয়ন কথা "কথা"
```
## কথা.**সূচক(*ক*, *সূচক*)** {.stdfunc}


এই কাজটি কথারাশি `ক` একটি নির্দিষ্ট অবস্থানের (`সূচক` অবস্থান) গ্রাফিম বা অক্ষর ফেরত দেয়। 

|প্রেরণমান|ধরন|বিবরণ|
|------|--|-----
|ক|কথারাশি|মূল কথারাশি যেখান থেকে অক্ষর খোঁজা হবে|
|সূচক|সংখ্যা (ধনাত্মক পূর্ণসংখ্যা)|অক্ষরের সূচক|
{.args-table}

|ফেরতমানের ধরন|বিবরণ|
|-----------|----|
|কথারাশি|`সূচক` অবস্থানে থাকা অক্ষর|
{.return-table}

###### **উদাহরণ** {.dh3}
```pankti
আনয়ন কথা "কথা"

ধরি শব্দ = "পলাশ"

?কথা.সূচক(শব্দ, ০)  // ফলাফলঃ প
?কথা.সূচক(শব্দ, ১)  // ফলাফলঃ লা
?কথা.সূচক(শব্দ, ২)  // ফলাফলঃ শ
```
## কথা.**ভাগ(*ক*, *বিভাজক*)** {.stdfunc}
এই কাজ একটি কথারাশি `ক` কে `বিভাজক` কথারাশির দ্বারা ভাগ করে একটি কথারাশিযুক্ত তালিকা ফেরত দেয়। 

|প্রেরণমান|ধরন|বিবরণ|
|------|--|-----
|ক|কথারাশি|মূল কথারাশি যেটিকে ভাগ করা হবে|
|বিভাজক|কথারাশি|কথারাশি যা দিয়ে `ক` কথারাশিকে ভাগ করা হবে|
{.args-table}

|ফেরতমানের ধরন|বিবরণ|
|-----------|----|
|তালিকা|এক বা একাধিক কথারাশি বিশিষ্ট তালিকা|
{.return-table}

###### **উদাহরণ** {.dh3}
```pankti
আনয়ন কথা "কথা"

ধরি বাক্য = "পঙক্তি,প্রোগ্রামিং,ভাষা"
ধরি বিভাজিত_বাক্য = কথা.ভাগ(বাক্য, ",")

?বিভাজিত_বাক্য[০]   // ফলাফলঃ পঙক্তি
?বিভাজিত_বাক্য[১]   // ফলাফলঃ প্রোগ্রামিং
?বিভাজিত_বাক্য[২]   // ফলাফলঃ ভাষা
```
## কথা.**পরিবর্তন(*মান*)** {.stdfunc}
এই কাজ `মান` প্রেরণমানে দেওয়া যেকোনো ধরনের রাশিকে কথারাশিতে পরিবর্তন করে ফেরত দেয়।


|প্রেরণমান|ধরন|বিবরণ|
|------|--|-----
|মান|যেকোনো রাশি|যে রাশিকে কথারাশিতে পরিবর্তিত করা হবে|
{.args-table}

|ফেরতমানের ধরন|বিবরণ|
|-----------|----|
|কথারাশি|`মান` -এর কথারাশিতে পরিবর্তিত রূপ|
{.return-table}

###### **উদাহরণ** {.dh3}
```pankti
আনয়ন কথা "কথা"

ধরি রাশি_মান = ১০১
ধরি কথা_মান = কথা.পরিবর্তন(রাশি_মান)

?প্রকার(রাশি_মান)         // ফলাফলঃ সংখ্যা
?প্রকার(কথা_মান)         // ফলাফলঃ কথারাশি
?রাশি_মান == কথা_মান     // ফলাফলঃ মিথ্যা
?কথা_মান              // ফলাফলঃ ১০১

রাশি_মান = [১,২,৩]
কথা_মান = কথা.পরিবর্তন(রাশি_মান)

?প্রকার(রাশি_মান)         // ফলাফলঃ তালিকা
?প্রকার(কথা_মান)         // ফলাফলঃ কথারাশি
?রাশি_মান == কথা_মান     // ফলাফলঃ মিথ্যা
?কথা_মান              // ফলাফলঃ [১, ২, ৩]

কাজ একটি_কাজ()
    ফেরাও সত্যি
শেষ

কথা_মান = কথা.পরিবর্তন(একটি_কাজ)

?প্রকার(একটি_কাজ)        // ফলাফলঃ কাজ
?প্রকার(কথা_মান)         // ফলাফলঃ কথারাশি
?কথা_মান              // ফলাফলঃ <কাজ 'একটি_কাজ'>
```
