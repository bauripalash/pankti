# সত্যমান (Booleans)

সত্যমানগুলি মূলত `যদি..নাহলে`, `যতক্ষণ..করো` ইত্যাদি ক্ষেত্রে মূলত ব্যবহৃত হয়।

| Boolean | Value |
|---|---|
| True | sotto, true, সত্যি |
| False |  false, mittha, মিথ্যা |

সত্যমান এভাবে লিখেও যেমন পাওয়া যায় তেমনই অন্য কিছু কাজের থেকেও ফেরত পাওয়া যায়। যেমন `১০০>৯৯` হল সত্য এবং `৯৯<১০০` হল মিথ্যা।
