# সংখ্যা
পঙক্তি প্রোগ্রামিং ভাষা সহ সব প্রোগ্রামিং ভাষাতেই সংখ্যা একটি গুরুত্বপূর্ণ অংশ। পঙক্তিতে বাংলা, ইংরেজি এবং দুটির মিশ্রণেও সংখ্যাগুলি লেখা যায়।

```pankti
১২৩.৪৫
123.45
১২৩.45
123.৪৫
1২3.৪5
```
উপরের উদাহরণে দেখানো সব সংখ্যাগুলিই পঙক্তিতে বৈধ। 

সংখ্যাগুলির মধ্যে যোগ, বিয়োগ, গুন, ভাগ, এবং ঘাত গণনা করা যায়। উদাহরণঃ 
#### যোগ
```pankti
১০০ + ২০০ // উত্তরঃ ৩০০
১২৩.৪৫৬ + ১.০০১ // উত্তরঃ ১২৪.৪৫৭
```

#### বিয়োগ
```pankti
১০০ - ২০০ // উত্তরঃ -১০০
১২৩.৪৫৬ - ১.০০১ // উত্তরঃ ১২২.৪৫৫
```

#### গুন
গুন করার জন্য `*` এই চিহ্নটি ব্যবহার করা হয়। কিবোর্ডে এটি সহজেই খুঁজে পাবে। 

```pankti
১০০ * ২০০ // উত্তরঃ ২০০০০
১২৩.৪৫৬ * ১.০০১ // উত্তরঃ ১২৩.৫৭৯৪৫৬
```
#### ভাগ
ভাগ করার জন্য `/` চিহ্নটি ব্যবহার করা হয়।

```pankti
১০০ / ২০০ // উত্তরঃ ০.৫
১২৩.৪৫৬ / ১.০০১ // উত্তরঃ ১২৩.৩৩২৬৫৭
```

#### ঘাত

\\( ক^খ \\) = ক ** খ

```pankti
২ ** ৩ // উত্তরঃ ৮
```
