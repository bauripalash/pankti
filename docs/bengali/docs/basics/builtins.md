# সাধারণ কাজসমূহ
## **দেখাও(*..*)** / **dekhao(*..*)** / **show(*..*)** {.stdfunc}
এই কাজ সমস্ত প্রেরণমানগুলি পরপর Terminal -এ প্রদর্শন করে। প্রতিটি প্রেরণমানের পর একটি স্পেস দিয়ে তারপর পরের প্রেরণমানটি দেখানো হয়।

>[!TIP]
> **বিঃদ্রঃ** এই কাজটি `?` বিবৃতির মতো লাইনের শেষে নতুন লাইন চিহ্ন (`\n`) যোগ করে না। এই বিষয়ে নীচে উদাহরণ দেওয়া আছে
>

|ফেরতমানের ধরন|বিবরণ|
|-----------|----|
|নিল|-|
{.return-table}

###### **উদাহরণ** {.dh3}
```pankti
দেখাও(1, 2, "তিন") // এইটি চালালে টার্মিনালে দেখাবে `1 2 তিন`
``` 
একাধিক `দেখাও(...)` একই লাইনে রাশিগুলি দেখাতে পারে,যেমন
```pankti
দেখাও(1)
দেখাও(2)
দেখাও("তিন")
``` 

এই উদাহরণটি চালালেও আগের রাশিগুলিই দেখাবে কিন্তু রাশিগুলির পর কোন স্পেস দেখা যাবে না। 
```
১২তিন
```

একটি দেখাও কাজের পর নতুন লাইন যোগ করতে গেলে কথারাশির বিশেষ চিহ্ন `\n` ব্যবহার করতে হবে। 
```pankti
দেখাও(1, "\n")
দেখাও(2, "\n")
দেখাও("তিন\n")
``` 
এটি চালালে এইরকম দেখাবেঃ 
```
১
২
তিন
```

শেষের বিবৃতিতে দ্বিতীয় রাশি হিসাবে `\n` যোগ করা হয়নি, যেহেতু `"তিন"` একটি কথারাশি তাই তার সাথে `\n` জুড়ে দেওয়া যায়। `দেখাও("তিন\n")` এবং `দেখাও("তিন", "\n")` এই দুটির ফলাফল একই দেখাবে।

## **প্রকার(*ক*)** / **prokar(*ক*)** / **type(*ক*)** {.stdfunc}
এই কাজটি `ক` কি প্রকারের রাশির তা কথারাশি হিসাবে ফেরত দেয়, যেমন ক যদি একটি সংখ্যা হয় তাহলে এটি ফেরত দেবে `সংখ্যা`, যদি একটি কথারাশি হয় তাহলে ফেরত দেবে `কথারাশি`।
|প্রেরণমান|ধরন|বিবরণ|
|------|--|----|
|ক|যেকোনো রাশি|যে রাশির প্রকার বা ধরন জানাতে চাওয়া হচ্ছে|
{.args-table}


|ফেরতমানের ধরন|বিবরণ|
|-----------|----|
|কথারাশি|`ক` রাশির ধরন। সম্ভাব্য মানগুলি হলঃ `সংখ্যা`, `সত্যমান`, `নিল`, `কাজ`, `তালিকা`, `ছক`, `কথারাশি`, `সাধারণ কাজ`, `উৎস`|
{.return-table}

###### **উদাহরণ** {.dh3}
```pankti
আনয়ন গণিত "গণিত"

কাজ একটি_কাজ()
    ফেরাও "একটি_কাজ"
শেষ

?প্রকার(১০০) // ফলাফলঃ সংখ্যা
?প্রকার(সত্যি) // ফলাফলঃ সত্যমান
?প্রকার(নিল) // ফলাফলঃ নিল
?প্রকার("পঙক্তি") // ফলাফলঃ কথারাশি
?প্রকার([1, 2, ৩]) // ফলাফলঃ তালিকা
?প্রকার({"নাম" : "পঙক্তি"}) //ফলাফলঃ ছক
?প্রকার(একটি_কাজ) //ফলাফলঃ কাজ
?প্রকার(গণিত) //ফলাফলঃ উৎস
?প্রকার(দেখাও) //ফলাফলঃ সাধারণ কাজ
```
## **আয়তন(*ক*)** / **ayoton(*ক*)** / **len(*ক*)** {.stdfunc}
এটি `ক` রাশির আয়তন ফেরত দেয়। তালিকার ক্ষেত্রে তালিকার উপাদান সংখ্যা, ছকের ক্ষেত্রে সূচকঃমান যুগল সংখ্যা এবং কথারাশির ক্ষেত্রে দৃশ্যমান অক্ষরের সংখ্যা ফেরত দেয়
|প্রেরণমান|ধরন|বিবরণ|
|------|--|----|
|ক|তালিকা / ছক / কথারাশি|যে রাশির আয়তন বা সেটিতে উপস্থিত উপাদানের সংখ্যা জানতে চাওয়া হচ্ছে|
{.args-table}

|ফেরতমানের ধরন|বিবরণ|
|-----------|----|
|সংখ্যা|-|
{.return-table}

###### **উদাহরণ** {.dh3}
```pankti
?আয়তন("পলাশ") // ফলাফলঃ ৩
?আয়তন([১, ২, ৩, ৪]) // ফলাফলঃ ৪
?আয়তন({"নাম" : "পলাশ", 1 : 2, ৩ : ৪}) // ফলাফলঃ ৩
```
## **সংযোগ( *তালিকার_নাম*, *উপাদান* )** / **songjog( *তালিকার_নাম*, *উপাদান* )** / **append( *তালিকার_নাম*, *উপাদান* )** {.stdfunc}
এই কাজটি `তালিকার_নাম` তালিকাতে `উপাদান` প্রেরণমান তালিকার একেবারে শেষে যোগ করে এবং তালিকার নতুন আয়তন ফেরত দেয়।
|প্রেরণমান|ধরন|বিবরণ|
|------|--|----|
|তালিকার_নাম|তালিকা|যে তালিকার শেষ উপাদান যোগ করা হবে|
|উপাদান|যেকোনো রাশি|যে উপাদান তালিকাতে যোগ করা হবে|
{.args-table}

|ফেরতমানের ধরন|বিবরণ|
|-----------|----|
|সংখ্যা|তালিকার বর্তমান উপাদান সংখ্যা|
{.return-table}

###### **উদাহরণ** {.dh3}
```pankti
ধরি একটি_তালিকা = [1, 2, "তিন"]
?সংযোগ(একটি_তালিকা, ৩) // ফলাফলঃ ৪ (তালিকার নতুন রূপঃ [1, 2, তিন, ৩])
?সংযোগ(একটি_তালিকা, ৪) // ফলাফলঃ ৫ (তালিকার নতুন রূপঃ [1, 2, তিন, ৩, ৪])
?একটি_তালিকা // ফলাফলঃ [১, ২, তিন, ৩, ৪]
```
## **গোলমাল(*বার্তা*)** / **golmal(*বার্তা*)** / **error(*বার্তা*)** {.stdfunc}
এই কাজটি `বার্তা` প্রেরণমানে দেওয়া কথারাশি বার্তা হিসাবে গোলযোগ দেখিয়ে উদাহরণটি চলা বন্ধ হয়ে যাবে।
|প্রেরণমান|ধরন|বিবরণ|
|------|--|----|
|বার্তা|কথারাশি|যে বার্তা গোলমালে দেখানো হবে|
{.args-table}
|ফেরতমানের ধরন|বিবরণ|
|-----------|----|
|নিল|-|
{.return-table}
###### **উদাহরণ** {.dh3}
```pankti
?১০০
গোলমাল("বন্ধ হোক")
?২০০
```
এই উদাহরণটি চালালে বার্তা দেখিয়ে উদাহরণ চলা বন্ধ হয়ে যাবে এবং টার্মিনালে এইরকম দেখাবে
```
১০০
Runtime Error: বন্ধ হোক
  2 | গোলমাল("বন্ধ হোক")

in <script> (a.pn) at 2:14
```
## **প্রেরণমান()** / **preronman()** / **args()** {.stdfunc}
এই কাজটি নির্দেশলিপি চালানোর সময় সেখানে দেওয়া `Command-line arguments` গুলি তালিকা হিসাবে ফেরত দেয়। নির্দেশলিপি যদি এইভাবে চালানো হয়, এই ক্ষেত্রে `Command-line arguments` গুলি হল 1, 2 এবং 3
```bash
pankti example.pn 1 2 3
```

|ফেরতমানের ধরন|বিবরণ|
|-----------|----|
|তালিকা|এক বা একাধিক কথারাশি বিশিষ্ট তালিকা|
{.return-table}
###### **উদাহরণ** {.dh3}
```pankti 
?প্রেরণমান()
```
এই নির্দেশলিপি যদি সাধারণভাবে চালানো হয় এইভাবে `pankti example.pn` তখন ফলাফল দেখাবে 
```
[]
```

আবার যদি এইভাবে চালাই `pankti example.pn hello 1 2` তাহলে এইরকম ফলাফল দেখাবেঃ
```
[hello, 1, 2]
```

## **পড়ো( *বার্তা* )** / **poro( *বার্তা* )** / **read( *বার্তা* )** {.stdfunc}
এই কাজ কনসোল বা টার্মিনাল থেকে তথ্য গ্রহণ করে যতক্ষণ না কিবোর্ডের `Enter` বোতাম চাপা হচ্ছে এবং সেই তথ্য কথারাশি হিসাবে ফেরত।
|প্রেরণমান|ধরন|বিবরণ|
|------|--|----|
|বার্তা|কথারাশি|যে বার্তার সাথে ব্যবহারকারী থেকে তথ্য চাওয়া হবে|
{.args-table}
|ফেরতমানের ধরন|বিবরণ|
|-----------|----|
|কথারাশি|ব্যবহারকারীর দ্বারা প্রদত্ত তথ্য, (নতুন লাইন চিহ্ন `\n` ব্যাতিত)|
{.return-table}
###### **উদাহরণ** {.dh3}
```pankti
ধরি নাম = পড়ো("নামঃ ")
?"আমার নাম হল " + নাম
```
এই উদাহরণটি চালালে টার্মিনালে এইরকম একটি বার্তা দেখাবে তারপর কোন কিছু লেখার পর Enter বোতাম চাপার পর দ্বিতীয় লাইনটি চলবে এবং লেখা উঠবে `আমার নাম হল <টাইপ করা নাম>`

প্রথমে এইরকম দেখাবে
```sh
নামঃ 
```

তারপর কোন কিছু লেখা হবে
```sh
নামঃ পলাশ বাউরি
```

এরপর Enter বোতাম চাপ দিলে এইরকম উত্তর দেখাবে

```sh
নামঃ পলাশ বাউরি
আমার নাম পলাশ বাউরি
```
