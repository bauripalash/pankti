# লেখা (String)

দুটি যুগল উদ্ধৃতি চিহ্ন (quotation mark) এর মধ্যবর্তী যেকোনো অক্ষর সমুহ লেখা বা স্ট্রিং বলে গণ্য হবে।

যেমন, 
```pankti
"পলাশ"
"ধরি"
"৩.১৪"
"+"
```
এগুলি সব স্ট্রিং।

লেখার ভেতর বিশেষ কিছু অক্ষর বা অক্ষরসমূহ ব্যবহার করে কিছু আকর্ষণীয় জিনিস করা যায়। 
যেমন একটি লেখার ভেতর যদি `\n` লেখা হয় তাহলে তা উত্তর  দেখানোর সময় একটি লাইনটিকে ভেঙে দেবে। যেমনঃ 
```pankti
দেখাও("পলাশ\nবাউরি")
```
এটি লিখে চালালে দেখা যাবে এটি উত্তর দেবে
```bash
পলাশ
বাউরি
```
একইরকম ভাবে, আরও কয়েকটি বিশেষ লেখা হল, `\a`, `b`, `\f`, `\r`, `\t`, `\v` ইত্যাদি। 

* `\a` থেকে কনসোলে একটি হালকা শব্দ করবে। 
* `\b` দিলে আগের অক্ষরটি মোছা হয়ে যাবে, যদিও এটা শুধু ASCII অক্ষরের ক্ষেত্রে কাজ করবে। উদাহরণঃ
```pankti
দেখাও("hel\blo")
```
এটি চালালে, এই উত্তর দেখাবে
```bash
helo
```
* `\f` যদিও আজকাল এটি বেশি ব্যবহৃত হয় না। এটাকে *Form Feed* বলে। উদাহরণঃ
```pankti
দেখাও("hel\flo")
```
এটি চালালে, এই উত্তর দেখাবে
```bash
hel
   lo
```
* `\r`, উইন্ডোজে নতুন লাইনের জন্য `\n` এর বদলে `\r\n` ব্যবহার করা হয়, যদিও উইন্ডোজে শুধু `\n` -ও কাজ করে। 
* `\t`, এটি দিয়ে Tab যোগ করা যায়। যেমনঃ
```pankti
দেখাও("পলাশ\tবাউরি")
```
এটি চালালে, এই উত্তর দেখাবে

```bash
পলাশ     বাউরি
```

* `\v` হল `\n\t` এর একধরনের সংক্ষিপ্ত রূপ। যেমনঃ
```pankti
দেখাও("পলাশ\vবাউরি")
```
এটি চালালে, এরকম দেখাবে

```bash
পলাশ
   বাউরি
```

এগুলি ছাড়া আরও তিন ধরনের বিশেষ অক্ষরসমূহ আছে `\xAA`, `\uAAAA`, `\UAAAAAAAA`

* `\x` দিয়ে যেকোনো Hex সংখ্যার দ্বারা অক্ষর লেখা যাবে, নীচের ASCII Table ছক একটা সংক্ষিপ্ত উদাহরণ যেগুলো ব্যবহার করা যায়। তবে আরও অনেক কিছু করা যাবে। উদাহরণঃ

```pankti
দেখাও("\x50\x41\x4c\x41\x53\x48") 
```
এটি চালালে দেখা যাবে। 

```bash
PALASH
```

> [!INFO]
> ![](https://upload.wikimedia.org/wikipedia/commons/1/1b/ASCII-Table-wide.svg)




* `\u` দিয়ে যেকোনো ইউনিকোড অক্ষর লেখা যাবে যেমন ইমোজি, অন্য ভাষার অক্ষর লেখা যাবে। \u এর পর ৪টি ইংরেজি অক্ষর লেখা যাবে, যেগুলি মিলে ইউনিকোড অক্ষর তৈরি হবে
যেমনঃ 

```pankti
দেখাও("\u2728")
```
এটি চালালে এই ইমোজি দেখাবে
```bash
✨
```
* `\U` দিয়ে যেকোনো ইউনিকোড অক্ষর লেখা যাবে যার কোড `0xFFFF` বা `65535` এর বেশি। মানে আজ পর্যন্ত তৈরি সমস্ত ইউনিকোড অক্ষর লেখা যাবে। \u এর পর ৮টি ইংরেজি অক্ষর লেখা যাবে, যেগুলি মিলে ইউনিকোড অক্ষর তৈরি হবে।  যেমনঃ 

```pankti
দেখাও("\U0001F680")
```
এটি চালালে এই ইমোজি দেখাবে
```bash
🚀
```

> [!NOTE]
> ### মনে রাখা দরকার
> তবে কোন পরিবর্তন ছাড়া এগুলিকে এই চিহ্নগুলিকে দেখানোর জন্য এভাবে লিখতে হবে `\\n`, `\\u`...

##  যোগ
দুটি লেখাকে `+` দিয়ে যোগ করা যায়। যেমনঃ 
```pankti
"পলাশ" + "বাউরি"
```
`পলাশ` এবং `বাউরি` দুটি লেখা একসাথে যুক্ত হয়ে একটি নতুন লেখা `পলাশ বাউরি` তৈরি করবে।
