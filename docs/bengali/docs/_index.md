# পঙক্তি প্রোগ্রামিং ভাষা

## পরিচিতি:
প্রোগ্রামিং ভাষা হল একটি বাংলা প্রোগ্রামিং ভাষা। পঙক্তি দিয়ে বাংলা ভাষা দিয়ে কম্পিউটার সফটওয়্যার তৈরি করা। গেম থেকে শুরু করে ১ + ২ গণনা করা যায়।

## তথ্যের বিবিধতা:
#### Numbers/সংখ্যা: 
```pankti
ধরি উদাহরণ = ১০০
উদাহরণ = ১২৩৪৫67890
উদাহরণ = 3.14
```

#### Strings/লেখা: 
```pankti
ধরি উদাহরণ = "আমার নাম পলাশ বাউরি"
উদাহরণ = "পঙক্তি প্রোগ্রামিং ভাষা"
উদাহরণ = "Hello World"
```

#### Booleans/বুলিয়ান:
```pankti
ধরি উদাহরণ = সত্যি
উদাহরণ = মিথ্যা
```

#### Nil(Null)/নিল
```pankti
ধরি উদাহরণ = নিল
```

#### HashTables/ছক: 
```pankti
ধরি উদাহরণ = {
    "নাম" : "পলাশ",
    "পদবী" : "বাউরি",
    "সময়" : ১১.১১
}
```

#### Array/তালিকা:
```pankti
ধরি উদাহরণ = ["পলাশ", "বাউরি", ১, ২, ৩, সত্যি]
```


### Variables/চলরাশি:
```pankti
ধরি নাম = "পলাশ"
ধরি পুরোনাম = নাম + " বাউরি"
ধরি মাস = "বৈশাখ"
ধরি সাল = ২০২৬
```

### Functions/কাজ:
```pankti
কাজ ঘুমানো(নায়ক)
    দেখাও(নায়ক + " এখন ঘুমোচ্ছে!")
শেষ
ঘুমানো("পলাশ")
```

### If...Else/যদি...তাহলে...নাহলে
```pankti
ধরি বয়স = ১০০
যদি বয়স >= ১৮ তাহলে
    দেখাও("সাবালক")
নাহলে
    দেখাও("নাবালক")
শেষ
```

### While...Do/যতক্ষণ...করো
```pankti
ধরি ক = ০
যতক্ষণ ক < ১০ করো
    দেখাও(ক)
    ক = ক + ১
শেষ
```

### Comments/মন্তব্য
```pankti
// এটি একটি মন্তব্য 
```
